Progressive Enhancement

Progressive Enhancement is using a programming language, page structure, and style information to make a website. HTML is what makes the website, CSS styles the html, and JavaScript implements features to the website. This is important because everyone has different devices and the way progressive enhancement is it helps users have the same functionality on their website.
